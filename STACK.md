# Technology Stack

## Backend
- **Runtime**: Node.js
- **Framework**: Express.js, tRPC
- **Language**: TypeScript
- **Database**: PostgreSQL (Prisma ORM)
- **Caching/Queues**: Redis
- **Validation**: Zod (Contract-First)

## Frontend
- **Framework**: TypeScript-based SPA
- **UI Library**: JUI (Custom Glassmorphism System)
- **Styling**: Tailwind CSS + Custom CSS Variables
- **State Management**: React/Vue Hooks

## Architecture
- **Monorepo**: Turbo
- **Contracts**: Shared Zod schemas (`@japavel/contracts`)
- **API Pattern**: tRPC (Type-Safe API) + REST (Legacy/External)
