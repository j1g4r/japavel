# Project Guidelines (Internal)

## CSS & Front-End Standards
- **Tailwind based RichUI**: High quality, modern components.
- **Smooth Transitions**: Kinetic/animated feel (AutoAnimate, Alpine, Framer Motion).
- **Vibrant Colors**: Premium, curated palettes (extended Tailwind colors).

## Architectural Protocol
- Zero-Stop Execution
- TDD & Scalability
- Domain-Driven Design (DDD) where applicable.
